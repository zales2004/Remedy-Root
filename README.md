REMEDYROOT 🌿

REMEDYROOT is a web-based medicinal plant identification system that uses Deep Learning and Computer Vision to recognize medicinal plants from leaf images. The system helps users identify plant species and access detailed information about their medicinal properties, therapeutic uses, useful plant parts, and regional availability.

Features
🌱 Medicinal plant identification from leaf images
🔍 Disease-based medicinal plant search
📖 Detailed medicinal information and therapeutic applications
📍 Information about plant distribution and availability
👥 User contribution module for adding new medicinal plants
🤖 Deep Learning-based image classification model
📱 Responsive web interface

Dataset

The model is trained using the Indian Medicinal Leaves Dataset containing 80 medicinal plant species with over 9,000 leaf images.
